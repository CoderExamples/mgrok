# MGROK —— 做更好的 Ngrok

--------------------------

#### Mgrok 客户端使用

* 下载 mgrok 客户端，请根据使用的操作系统，下载对应的版本

* 解压缩，并进入到解压后的目录，运行 mgrok

例如

**建立HTTP隧道**

```
mgrok -subdomain pub -proto=http 80
```

**建立数据库隧道**

```
mgrok -proto=tcp 3306
```

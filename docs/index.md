#### 使用本站提供的 Ngrok 服务

* 下载 ngrok 客户端，请根据使用的操作系统，下载对应的版本

* 解压缩，并进入到解压后的目录，运行 ngrok.exe

<br/>

例如

**建立HTTP隧道**

```
ngrok -subdomain pub -proto=http 80
```

**建立数据库隧道**

```
ngrok -proto=tcp 3306
```
